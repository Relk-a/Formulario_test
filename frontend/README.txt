# Formulario de Pago - Crosspay Solutions

Aplicación web para el registro y administración de pagos con tarjeta, con frontend en React y backend en Node.js/Express.

---

## Características

- **Formulario de pago**: Registro seguro de pagos con tarjeta.
- **Panel de administración**: Visualización de transacciones protegida por login.
- **Autenticación JWT**: Solo administradores pueden acceder al panel.
- **Modo claro/oscuro**: Interfaz adaptable.
- **Validaciones y enmascaramiento**: Manejo seguro de datos sensibles.
- **Alertas modales**: Mensajes de éxito y error visuales y modernos.
